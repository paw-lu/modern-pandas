# Tidy data takeaways

## Data

- `.read_html` to read html tables
- `.dropna` has a `thres` argument
- `.diff` can be used to find difference between adjacent rows.
  Can be used with `.gropuby`.

## Tidying data

- `pd.melt`
- `pd.pivot_table`
- `.stack` from wide to long
- `.unstack` from long to wide
